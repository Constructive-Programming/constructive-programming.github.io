Each library below is independently versioned at
[github.com/Constructive-Programming](https://github.com/Constructive-Programming).
They share a common motivation — making *correct by construction* practical
in production code — and each tackles a different slice.

{% include libraries.html %}

[cats-eo](https://github.com/Constructive-Programming/eo) is the flagship —
production-grade existential optics for Scala 3 with discipline-checked
laws and performance benchmarked against Monocle.
[scala-cardinality](https://github.com/Constructive-Programming/scala-cardinality)
and [surreals](https://github.com/Constructive-Programming/surreals) are
smaller explorations that back the methodology with code.
